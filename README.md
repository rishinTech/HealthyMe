# Healthy Me

Healthy Me is a responsive web application that enables users to calculate Body Mass Index (BMI), track daily calorie intake, and monitor progress toward personal fitness goals.

The application supports multi-unit BMI calculation (kg/lbs and cm/feet-inches), dynamic calorie aggregation, user-specific data persistence using LocalStorage, and a clean, modern interface optimized for both mobile and desktop devices.

## Tech Stack

HTML5  
CSS3  
JavaScript (ES6)  
LocalStorage API
